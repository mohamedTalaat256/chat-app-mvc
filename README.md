# chat-app-mvc
Realtime chat app created with spring MVC and websocket that provide conversation between users.

### dependencies:
##### 1- Thymeleaf
##### 2- Spring Security
##### 3- JWT
##### 4- Spring data JPA
##### 5- Hibernate Validatior
##### 6- Spring lombok
##### 7- Spring Websocket
